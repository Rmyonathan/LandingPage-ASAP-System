# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of Toko GlobalTeknik landing page
- Complete feature showcase with all application screenshots
- Responsive design for mobile and desktop
- SEO optimized with meta tags and structured data
- PWA support with service worker
- Smooth scrolling navigation
- Interactive animations and hover effects
- Contact form with validation
- Multi-platform deployment support (Vercel, Netlify, GitHub Pages)
- Performance optimizations
- Accessibility features
- Cross-browser compatibility

### Features
- **Sistem Akuntansi**: Buku besar, jurnal umum, neraca saldo, laporan laba rugi
- **Manajemen Stok**: Master barang, adjustment stok, mutasi stok, konversi satuan
- **Manajemen Penjualan**: Faktur penjualan, sales order, surat jalan, retur
- **Multi Database**: Dukungan untuk 2+ database terintegrasi
- **Unlimited Account**: Akun dan role tanpa batas
- **Import Excel**: Kemudahan migrasi data dari Excel

### Technical
- HTML5 semantic markup
- Tailwind CSS for styling
- Vanilla JavaScript for interactivity
- Font Awesome icons
- Service Worker for offline support
- Manifest.json for PWA
- Robots.txt and sitemap.xml for SEO
- 404 error page
- Custom CSS animations
- Form validation
- Scroll indicators
- Lazy loading support
