# Toko GlobalTeknik - Landing Page

Landing page untuk sistem akuntansi terintegrasi Toko GlobalTeknik yang mendukung manajemen toko, stok, dan penjualan dengan multi-database.

## Fitur Utama

- **Sistem Akuntansi Lengkap**: Buku besar, jurnal umum, neraca saldo, laporan laba rugi
- **Manajemen Stok**: Master barang, adjustment stok, mutasi stok, konversi satuan
- **Manajemen Penjualan**: Faktur penjualan, sales order, surat jalan, retur
- **Multi Database**: Dukungan untuk 2+ database terintegrasi
- **Unlimited Account**: Akun dan role tanpa batas
- **Import Excel**: Kemudahan migrasi data dari Excel

## Teknologi

- HTML5
- Tailwind CSS
- Font Awesome Icons
- Vanilla JavaScript

## Deployment ke Vercel

### Metode 1: Deploy via Vercel CLI

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Login ke Vercel:
```bash
vercel login
```

3. Deploy:
```bash
vercel
```

### Metode 2: Deploy via GitHub

1. Push kode ke repository GitHub
2. Login ke [Vercel](https://vercel.com)
3. Import project dari GitHub
4. Deploy otomatis

### Metode 3: Drag & Drop

1. Login ke [Vercel](https://vercel.com)
2. Drag folder `landing-page` ke dashboard Vercel
3. Deploy otomatis

## Struktur File

```
landing-page/
├── index.html          # Halaman utama
├── package.json        # Konfigurasi npm
├── vercel.json         # Konfigurasi Vercel
└── README.md          # Dokumentasi
```

## Custom Domain

Setelah deploy, Anda dapat menambahkan custom domain di dashboard Vercel:
1. Masuk ke project settings
2. Pilih tab "Domains"
3. Tambahkan domain Anda
